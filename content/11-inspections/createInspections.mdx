# Добавление новых медицинских осмотров

<div style="text-align: justify;">

Если у Вас есть доступ к учетной записи администратора, вы можете создавать новый шаблон осмотров, который характеризует последовательность этапов обслуживания (с созданием этапов можно ознакомиться [тут](/)), необходимых для выдачи заключения по определенным факторам (согласно [приказу Минздрава России от 28.01.2021 N 29н](https://www.consultant.ru/document/cons_doc_LAW_375353/f0e59583535b88589c7fb500d5479e6fd51fd350/)), либо без них, например, для водителей различных категорий. Чтобы создать шаблон осмотра, нажмите на вкладку "<strong>Осмотры</strong>" в левой панели приложения, как показано на рисунке 22. После нажатия откроется окно со списком существующих осмотров и возможностью добавления новых.

</div>

<div style="flex: 1;">
  <ImageModal src="/images/image22.png" alt="Окно мед осмотров" />
  <p style="text-align: center; font-size: 14px;">Рисунок 22 - Окно медицинских осмотров.</p>
</div>

<div style="text-align: justify;">
  Нажмите на кнопку (расположенную сверху справа на рисунке 22) "<strong>Добавить шаблон</strong>".
  В появившемся окне, изображённом на рисунке 23, есть следующие поля:
</div>

<div style="text-align: justify;">

- **Наименование** - здесь Вы записываете произвольное наименование создаваемого шаблона. Например, если Вам необходим осмотр по фактору [1.7.2 Бороводороды(О)](https://www.consultant.ru/document/cons_doc_LAW_375353/c54bf6a9b79ae804dacade649b421009b6b98ae9/), то одним из вариантов названия будет "1.7.2". Данное имя будет отображаться в поле "**ОСМОТРЫ**" раздела [Отслеживание осмотров](/11-inspections/inspectionTracking). Поле является обязательным.
- **Фактор применения** - если Вы настраиваете шаблон для выдачи заключения по определенным [факторам](https://www.consultant.ru/document/cons_doc_LAW_375353/f0e59583535b88589c7fb500d5479e6fd51fd350/), то в выпадающем меню выберите необходимые. Иначе, оставьте поле пустым.
- **Начальный этап** - укажите услуги, которые пациент должен пройти в первую очередь. В выпадающем списке указаны ВСЕ услуги, которые созданы в разделе **Услуги** (с созданием новых услуг можно ознакомиться [тут](/)). Данное поле является не обязательным.
- **Промежуточные этапы** - любые услуги, которые необходимо предоставить перед завершающим этапом осмотра. В выпадающем списке указаны ВСЕ услуги, которые созданы в разделе [Услуги](/). Данное поле является не обязательным.
- **Завершающий этап** - последняя услуга, необходимая для выдачи заключения. Данное поле является не обязательным.

Заполнив обязательные поля, Вы можете нажать кнопку "<strong>Добавить</strong>", после чего в системе появится новый шаблон медицинского осмотра.

</div>
